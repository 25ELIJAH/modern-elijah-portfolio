# Elijah James Portfolio Website - Premium Structure V2.0

This project has been organized as a feature-based React portfolio. Each major website section lives under `src/features`, shared content lives in `src/shared/data`, and reusable layout/helpers live in `src/shared`.

## Folder Map

- `src/features/home` - animated hero, portrait area, dynamic text, brand statement, CTAs, social links, scroll indicator
- `src/features/trust-bar` - experience, projects, clients, companies, and technology statistics
- `src/features/about` - biography, story, mission, vision, career timeline, education, certifications, values
- `src/features/services` - development, marketing, content, design, branding, and consulting services
- `src/features/stacks` - software development stack and digital marketing stack
- `src/features/tools-showcase` - hover-ready tool cards, descriptions, experience levels, and usage stats
- `src/features/projects` - featured projects and project detail entry points
- `src/features/case-studies` - problem, strategy, implementation, outcome, and feedback structure
- `src/features/experience` - interactive timeline-ready work experience
- `src/features/certifications` - professional certifications, ICT qualifications, workshops, training, and badges
- `src/features/achievements` - milestones, awards, recognition, campaigns, and major projects
- `src/features/testimonials` - reviews, ratings, video testimonial placeholders, and success stories
- `src/features/brands` - organizations, client logos, partner logos, and collaboration showcase
- `src/features/blog` - digital marketing articles, SEO guides, development tutorials, trends, and insights
- `src/features/resources` - CV, portfolio PDF, company profile, and rate card downloads
- `src/features/booking` - service selection, date, time, consultation type, notes, and confirmation email flow
- `src/features/contact` - WhatsApp, phone, email, social links, and contact form
- `src/features/faq` - common, pricing, timeline, and support questions
- `src/features/newsletter` - email subscription and updates
- `src/features/premium-features` - premium feature checklist and implementation roadmap
- `src/features/layout` - sticky navigation and footer
- `src/shared/data` - centralized portfolio content arrays
- `src/shared/components` - reusable section wrappers and shared UI
